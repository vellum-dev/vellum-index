# Vellum Package Index

Web-based package browser for the [Vellum](https://github.com/vellum-dev/vellum) package manager.

## Development

```bash
npm install
npm run dev
```

## Data Sync

Package metadata is pushed from the [vellum](https://github.com/vellum-dev/vellum) repo via CI. Each push triggers an automatic build and deploy of this site.
